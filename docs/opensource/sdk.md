## 获取SDK中所需的信息

### environment key

1. 点击左侧菜单中"账户管理"
2. 切换到Tab "项目"
3. 找到对应的项目及环境，如 "Default Project" -> "Production"
4. 在环境名的右侧找到Secret，粘贴他即可

### 开关的key

1. 点击左侧菜单中"开关列表"
2. 找到对应的开关，点击进入开关管理页面
3. 切换到Tab "设置"，找到"Key Name", 粘贴其内容即可
